使用GitHub来备份hexo的源码

前提条件：已成功搭建完成并部署hexo

创建GitHub仓库

1.登录GitHub并创建仓库，设置为私密
2.在hexo根目录创建.gitignore文件，设置要忽略的文件或文件夹，填写不需要备份的文件和文件夹。

3.使用git备份源文件到GitHub

如需同时使用gitee

恢复

1.将备份的仓库克隆到本地

2.使用备份的 package.json 文件来恢复安装 Hexo 插件，使用 npm 进行安装：

这将读取 package.json 文件中的依赖项列表，并下载并安装所需的插件和模块。

3.生成静态网页并部署即可。
